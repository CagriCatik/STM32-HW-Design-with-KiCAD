## System bootloader

The system bootloader is a small program stored permanently inside the microcontroller by the manufacturer. It cannot be erased. It allows firmware upload using interfaces such as UART, USB DFU, CAN, or SPI depending on the model. When BOOT0 is set high, the MCU runs this loader instead of the user program.
